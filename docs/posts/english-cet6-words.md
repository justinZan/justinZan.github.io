# 英语六级词库

这页按六级水平整理核心词汇，格式保持和“英语高频单词”一致：单词、中文、例句。

说明：CET-6 词汇量很大，很多基础词和四级词会重复出现。这里先整理六级阅读、写作、听力中更常见、更值得优先掌握的核心词，后续可以继续按 A-Z 扩展。

## 学习方法

不要只背中文意思。建议每次选 20 个词，先读例句，再把例句改成自己的句子。
