# 英语常见短句

这页整理日常和工作中最容易用到的英文短句。先把短句说顺，再慢慢替换关键词。

| --- | --- |

| Nice to meet you. | 很高兴认识你。 |
| Long time no see. | 好久不见。 |
| How was your day? | 今天过得怎么样？ |
| Do you have a minute? | 你现在有空吗？ |
| Can I ask you something? | 我可以问你个问题吗？ |
| Sorry to bother you. | 抱歉打扰你。 |

| Good afternoon. | 下午好。 |
| How have you been? | 你最近怎么样？ |

| Is now a good time? | 现在方便吗？ |

| Let's get started. | 我们开始吧。 |
| I'll keep it short. | 我会简短说明。 |

## 表达问题和风险

| English | 中文 |
